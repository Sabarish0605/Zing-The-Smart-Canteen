ℹ️ Default Meals System - Client-Side Only

🔒 Default Meals (Auto-Generated by Student App):

  • Veg Fried Rice = Rice (1) + Veg (1) - Stock: min(rice, veg)
  • Veg Fried Noodles = Noodles (1) + Veg (1) - Stock: min(noodles, veg)
  • These meals are NOT stored in vendor database
  • Student app generates them dynamically from raw materials
  • Vendor CANNOT edit them - only update raw materials

🍳 Add-ons (Optional for Default Meals Only):

  • Students can add: Egg (+₹15) or Chicken (+₹30)
  • Veg is ALWAYS included automatically (no checkbox, no charge)
  • Add-ons reduce stock ONLY when selected

📊 Example Flow:

  • Rice = 10, Veg = 8 → Student sees "Veg Fried Rice: 8 available"
  • Student orders 1 Rice + Egg → Backend deducts: Rice-1, Veg-1, Egg-1
  • New stock: Rice=9, Veg=7, Egg reduces → Next student sees 7 available
